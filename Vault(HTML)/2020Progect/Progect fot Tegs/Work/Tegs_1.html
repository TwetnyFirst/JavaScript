Тег "p" - служит для параграфа(срока начинается с новой строки)

Tег "div" служит как опора для сайта,это блочный тег

Тег "header" - для верха страницы

Тег "nav" - для навигации

Tег "span" - строчный,служит для написания текста в скроку, и ЕГО МОЖНО поместить в блочный тег, А БЛОЧНЫЙ в СТРОЧНЫЙ нельзя

Tег "a",строчный, предназначен для создания ссылок.Для тега "a" есть атрибут "target", он служит для того чтобы ссылка открывалась в новом окне

Tег "img", для фотографий, атрибут "src" - для пути фотографии, атрибут "alt" - текст в этом атрибуте будет выведен , если фото не загрузится.

Тег "br" предназначен для перекидывания текста, стоящего после тега, на следующюю строку

    Есть два вида таблиц:

  1. Тег "ul" - маркерованные таблицы
  2. тег "ol" - нумерованные таблицы
  3. каждый из тегов таблиц должен быть заключен в тег "li"
Тег "footer" - для низа

теги h1-h6 пишут жирным текстом по убыванию соответсвенно

пример текста h1

Тег "form" - блочный тег, выступает в роли оболочки.
  • Атрибут "action" определяет куда будут отправлены данные, значение атрибута может быть адресс ссервера, но можно и поставить #.
  • Атрибут "enctype" нужет для передачи прикрепленных файлов,со значением "multipart/form-data"
  • Атрибут "method" - определяет метод передачи данных и имеет два значения:
    1. Значение get - по умолчанию, и передача данных идет в адресной строке
    2. Значение post - передаёт данные через запрос браузера, лечше подходит

Тег "input" служит для приёма данных от пользователя, Его атрибуты:

  • Атрибут "type" определяет тип,есть несколько значений
    1. Значение "text"
    2. Значение "tel"
    3. Значение "email"
    4. Значение "password"
    5. Значение "radio" - это значение нужно для выбора ОДНОГО из многих, и их лучше компелировать под одним name
      6. choose 1
      choose 2
    6. Значение "checkbox" - это значение нужно для выбора нескольких
      7. want this
      this
      and this
    7. Значение "file" - добавляет возможность прикрепить файл
  • Атрибут "name" даёт наименование нашему input
  • Атрибут "value" - предназначен для вывода значения
  • Атрибут "placeholder" добавляет в поле подсказку, что вводить в это поле
Тег "textarea" - предназначен для ввода объемного текста

Теги "select" + "option используются вместе, для создания выпадающего списка

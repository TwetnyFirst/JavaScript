Hello, this is page for CSS

CSS можно подключить к HTML файлу с помощью команды "link"

В CSS можно обращаться к:
  • Тегам
  • Классам
  • Опред.Тегам с опред.Классом

Свойсво "width" - регулирует ширину блока(например в этом блоке стоит ширина 200px).Это значение может быть в:
  • px
  • %
  • через "calc"

Свойство "height" - регулирует высоту блока
Свойство "backround" - определяет то,как быдет выглядить задний план начего блока,его можно задать с помощью:
  • Цвета - пишем backround-color и после него пишем "#" и цифры
  • Вставить изображение - пишемbackround-imageдалее "url" и ссылку
  • Чтобы подогнать,например, размер картинки под размер блока - пишем"backround-size" и далее применяем значение "cover"
  • "backround-position" - определяет то,на сколько нужно отступить в САМОЙ фотографии по X и пo Y
  • "backround-repeat" - нужен, чтобы после оступа с помощью"backround-position" фото не повторялась, этого можно достигнуть с помощью значения"no-repeat"
  • "backround-attachment" - фиксирует картику(или то что мы задали) на фоне благодаря значению"fixed"
Свойство"border" - создает границы нашему блоку и пишется по примеру: border:(толщину границ в px) (стиль границ) (цвет)

Есть несколько основных стилей:

  • solid
  • dashed
  • dotted
Одно из самых полезых подсвойств является "border-radius",в значение которого если записать 50%, то получится круг
Подствойство "rgba" - можно задать цвет,первыми тремя значениями, и прозрачность четвертым
Свойство "boxshadow" - добавляет тень нашему блоку.Первое значение это смещение тени по X,второе на Y,третье это размытие нашей тени и четвертое значение это радиус тени.
Свойство "font-size" - определяет величину текста на странице.Его можно задать в:
  • px -стандартный размер 16px
  • em - это относительная величина от пикселя и относится она к родительскому блоку,а чтобы она относилась к изначатьно заданной величине(16px) нужно дописать "r"(rem)

Свойтво "font-weight" - определяет толщину (жирность) текста, со значением bold

Свойство "font-style" - определяет курсивноть, со значением italic

Свойство "text-transform" - может сделать все буквы заглавными

Свойсто "text-decoration" - может подчеркнуть, зачеркнуть... текст

Шрифты(не изучено!!!)

Чтобы подключить шрифты, для начала их нужно скачать(обычно у них расширение ttf).
Далее чтобы подключить нужно прописать:
@font-face{
font-family:(название шрифта);
src: local("(название шрифта");
url(путь до шрифта);
}
Вот мы рассказали CSS, что есть такой шрифт и теперб его надо подключить к нашей странице: с помощью свойства "font-famity" и в значении указываем локальное название шрифта
Tекст изначально выравнивается по левому краю.Свойство CSS"text-align" - меняет это в зависимости от значения:
  • "center"
  • "left" - по умолчанию
  • "justufy" по ширине блока
  • "right"
  • "inherit" - пропустить свойство
Свойстово "line-height" - определяет межстрочное расстояние!!! если значение"line-height" = значению ширины"div"
Свойтво "margin" - задает отступы по осям X и Y, на зананное значение.
Также сужествуют приставки:
  1. "-top"
  2. "-left"
  3. "-right"
  4. "-bottom"
Либо тоже самое можно задать четырьмя значениями.
Свойтво "padding" - задаёт внутренние отступы и имеет приставки:
  1. "-top"
  2. "-left"
  3. "-right"
  4. "-bottom"
Свойство "float" -может распологать блоки с лева на право
Свойство "overflow" Со значением:
  • "hidden" - может спятать наш скролл
  • "auto" - модет добавить скролл даже в бокс

Изначально если у блока есть родительский блок, то "детский" блок будет перемещатся относительно ему.
Свойство CSS - "position" со значением :
  • "absolute" - может сделать наш блок, относительным body.
    После того как мы задали свойство absolute, начинают действовать свойства left, top ,right, bottom.
  • "relative" - становится родительским для блока со свойством - position:absolute
  • "fixed" - закрепляет наш блок в указанной нами точке и скрол на него влиять не будет
Свойство "cursor" - меняет курсов соответственно значениям, например: pointer

Псевдоклассы

Псевдоклассы - определяет изменения нашего блока, после некоторых манипуляций над ним и пишется после названия класса через ":"
Псевдоклассов бывает много, но вот некоторые из них для Тега "div":
  1. "hover" - изменяет наш блок после навода на него курсора
  2. "active" - изменяет наш блок после нажания на него
  3. "nth-child" - позволяет изменить определённый блок , номер которого мы указали в скобках.Если в скобках мы укажем например "2n" или "2n+1", то будут изменятся каждый четный и нечетный блок соответственно
  4. "before" - тот контент, который мы указали в before появится перед содержимым блока
    • Чтобы создать контент нужно воспользоваться значением "content" и в нео вписать какое-то значение
    • Либо можно воспользоваться значением "display:block" и вставить некоторый блок
  5. "after" - тоже что и before, только после содержимого
Некоторые из псевдоклассов для "input":
  • "focus" - если добавить backround, то при активиции инпута этот backround будет виден
Значение "none" - сбрасывает предыдущие значения свойств Значение"inherit".если записать это значение в свойство, то оно примет значение из родительского блока(если такой имеется).
Чтобы задать приоритет свойству нужно использовать"!important" после значения свойства.
